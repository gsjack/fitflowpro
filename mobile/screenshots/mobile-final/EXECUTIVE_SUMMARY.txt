🎓 KEY LEARNINGS
────────────────────────────────────────────────────────────────────────────
What Worked:
  ✅ Expo Dev Launcher navigation (not direct am start)
  ✅ File size validation (immediate quality check)
  ✅ Wait 3 seconds after launch (UI settlement)
  ✅ No input field taps before capture (avoids keyboards)

What Didn't Work:
  ❌ Direct app launch (requires Dev Launcher)
  ❌ Tab tap navigation (Register tab unresponsive)
  ❌ adb input text (concatenates instead of replacing)
  ❌ UI-based login (API connectivity issue)

Best Practices:
  1. Always verify backend reachable from emulator FIRST
  2. Never tap input fields before screenshot capture
  3. Check file sizes immediately after capture
  4. Restart app fresh for each screen
  5. Document blockers, don't waste time on unfixable issues
